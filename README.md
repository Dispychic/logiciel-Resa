# Dipsy Tables

Application locale de gestion de tables et réservations pour Dipsy.

## Mise en ligne GitHub Pages

1. Créer un nouveau repository GitHub.
2. Ajouter `index.html` et l'image PNG dans le repository.
3. Aller dans Settings > Pages.
4. Source: Deploy from a branch.
5. Branch: main, folder: /root.
6. Ouvrir le lien GitHub Pages généré.

Les données sont sauvegardées dans le navigateur de chaque appareil.
